# Flare.Components.RichTextEditor

Rich text (WYSIWYG) editor component for the [Flare](https://github.com/jrfrigat/Flare) Blazor
component library. Add-on package that extends `Flare.Components`.

```sh
dotnet add package Flare.Components.RichTextEditor
```

Requires `Flare.Components` and a `Flare.Theme.*` package. Use `<FlareRichTextEditor>` once Flare is
set up (see the `Flare.Components` readme).
