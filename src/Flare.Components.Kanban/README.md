# Flare.Components.Kanban

Kanban board component (draggable columns and cards) for the
[Flare](https://github.com/jrfrigat/Flare) Blazor component library. Add-on package that extends
`Flare.Components`.

```sh
dotnet add package Flare.Components.Kanban
```

Requires `Flare.Components` and a `Flare.Theme.*` package. Use `<FlareKanban>` once Flare is set up
(see the `Flare.Components` readme).
