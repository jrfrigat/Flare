# Flare.Components.Carousel

Carousel / slideshow component for the [Flare](https://github.com/jrfrigat/Flare) Blazor component
library. Add-on package that extends `Flare.Components`.

```sh
dotnet add package Flare.Components.Carousel
```

Requires `Flare.Components` and a `Flare.Theme.*` package. Use `<FlareCarousel>` once Flare is set up
(see the `Flare.Components` readme).
