# Flare.Core

Core abstractions and the theme service for the [Flare](https://github.com/jrfrigat/Flare) Blazor
component library: `IThemeService`, design tokens, theme/palette records and shared base types.

You normally don't reference this directly - it comes in transitively via `Flare` or
`Flare.Components`. Install it on its own only when building a Flare theme or extension package.

```sh
dotnet add package Flare.Core
```
