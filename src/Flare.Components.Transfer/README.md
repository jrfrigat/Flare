# Flare.Components.Transfer

Transfer / dual-list-box component for moving items between two lists, for the
[Flare](https://github.com/jrfrigat/Flare) Blazor component library. Add-on package that extends
`Flare.Components`.

```sh
dotnet add package Flare.Components.Transfer
```

Requires `Flare.Components` and a `Flare.Theme.*` package. Use `<FlareTransfer>` once Flare is set up
(see the `Flare.Components` readme).
